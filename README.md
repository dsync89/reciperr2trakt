# Reciperr2Trakt

A no-nonsense program to import Reciperr JSON list (StevenLu List) to your Trakt List. 

Make sure to generate API token from your Trakt account.

Replace the following variables:
- CLIENT_ID
- CLIENT_SECRET
- USERNAME
- LIST_NAME
- LIST_DESCRIPTION
- MOVIES_URL

Then run,

```
python3 app.py
```
